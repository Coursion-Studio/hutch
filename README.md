# Hutch

Multi-tool macOS utility from [Coursion Studio](https://github.com/Coursion-Studio).

**[Download the latest release →](https://github.com/Coursion-Studio/hutch/releases/latest)**

Requires macOS 15 (Sequoia) or later.

## What's inside

- **Window Manager** — keyboard-first window tiling and snapping
- **Stash** — quick shelf for files you're moving between apps
- **Toggles** — one-click system toggles (appearance, audio, more)
- **Browser Picker** — route http(s) links to the right browser by rule

Each tool is independently toggleable from Preferences.

## Updates

Hutch ships with [Sparkle](https://sparkle-project.org) — updates download silently in the background and apply on next quit. Update feed: [`hutch.coursion.studio/appcast.xml`](https://hutch.coursion.studio/appcast.xml).

## Privacy

Hutch runs entirely on your Mac. No analytics, no telemetry, no account.
