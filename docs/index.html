What's inside

What's in the drawer today.

Each tool earns its place. None of them are placeholders. More on the way.

Pounce — the command bar

⌘-Space and start typing. Apps, system actions, your toggles, files, recent downloads, browser tabs — every workflow in one keystroke. Math, currency, units, even AI prompts to your own provider. Like Raycast, except everything is local and BYO key.

  • App launcher (with switch & quit)
  • Window switcher
  • Clipboard history
  • Live FX + units calc
  • iOS simulator picker
  • AI · BYO Anthropic / OpenAI / Ollama
  • Web bangs (g, gh, w, npm, …)
  • Custom commands
  • Markdown preview

Window Manager

Snap windows to halves, quarters, thirds, or any ratio you like. Click a tile or hit a hotkey — they fire globally, in any app.

  • Halves & quarters
  • Thirds
  • Custom ratios
  • Global hotkeys
  • Multi-display

Stash

A floating shelf for files mid-drag. Drag from one app, drop on Stash, drag out into another. Or summon it on a hotkey when you need it.

  • Drag-and-drop shelf
  • Cursor-shake summon
  • Hotkey ⌃⌥⌘V
  • Pin items

Browser Picker

Set Hutch as your default browser and route every link by rule — work URLs to Chrome, personal to Safari, anything weird to a picker. Five seconds of setup, hours saved.

  • Per-domain rules
  • Picker prompt
  • App-source rules
  • Ordered fallbacks

Toggles

One panel for the macOS switches that hide three menus deep — Dark Mode, Stage Manager, Hot Corners, AirPods, Bluetooth. Click the menu bar icon, flip the switch.

  • Appearance
  • Stage Manager
  • Hot Corners
  • AirPods
  • Bluetooth

    Why Hutch

    Native. Private. Out of the way.

    Hutch is a paid utility's worth of tools — free, forever.

    100% native SwiftUI

    No Electron. No web views. Cold-starts in milliseconds, feels like macOS because it is macOS.

    Runs on your Mac

    No accounts, no telemetry, no uploads. Hutch's only network call is checking for app updates.

    Lives in the menu bar

    One tray icon, every tool. Hutch only joins the Dock and Cmd+Tab while the dashboard is open.

    Auto-updates

    Sparkle-signed updates ship over the air. Apple-notarized DMGs, EdDSA-verified — no App Store middleman.
